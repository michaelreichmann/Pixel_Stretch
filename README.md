# Pixel Stretch

*Max 8.3*

This Jitter Effect stretches pixels across a strip from two sides and interpolates between them.<br/> 
The controls are:<br/>
**position** of the strip<br/>
**size** of the strip<br/>
**x/y**: the axis along which the strip stretches<br/>
